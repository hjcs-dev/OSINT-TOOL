# Install Tool

1. Clone the repository:
```
git clone https://github.com/hjcs-dev/OSINT-TOOL
```
## Install dependencies
```
pip install -r requirements.txt
```
## Select the downloaded repository
```
cd OSINT-TOOL
```
# Run the code
```
py ./main.py
```
